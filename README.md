# 2020 104x梅竹黑客松

一零四資訊科技與梅竹黑客松合作，提供職缺及求職者行為資料，請參賽者發揮創意，打造多元創新服務。

## 活動網址
[依梅竹黑客松為主]

## 主題說明
隨著台灣社會邁入少子化高齡化，年輕人的供給快速減少，如何讓社會的中壯齡人才資源能有效利用，改變企業對中壯齡人才的認知，提供友善的媒合流程，讓社會能持續穩定發展，是 104人力銀行【壯有所用】的願景。 

請參賽者利用 104 提供的資料集或其他外部資源，發想一個最小可行性產品來滿足【壯有所用】的願景。
